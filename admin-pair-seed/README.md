# Admin Pair Seed

The public feedback board does not include an admin screen.
A separate admin web can be built later and paired with this project through the existing admin feedback API.

Planned admin pair:
- Separate repository / separate Netlify site
- Private ADMIN_KEY only in the admin site environment
- Read feedback from the feedback board API
- Export CSV and moderate feedback later

This keeps the public board simple and avoids mixing admin UI with user-facing pages.
